  • A work in progress, however all examples work with current Rakudo

Check it out

      
  $ git clone https://github.com/perl6/perl6-examples.git
  $ cd perl6-examples
  $ make html       # wait a bit...
  $ make webserver  # view examples as html http://localhost:3000
  $ make run-all    # run all examples
  $ make test       # run the test suite
